# Audio Latency Fixer

<p align="center">
  <img src="Assets/AudioLatencyFixer.svg" alt="Audio Latency Fixer Icon" width="512"/>
</p>

A lightweight Windows tool to analyze and improve system audio latency.

## Features

* Detects your current audio device
* Estimates system latency
* Applies safe system optimizations
* Recommends better hardware setups
* Runs in the system tray


## Installation

Go to the "Releases" page and download a version. Then run through the installer.

## Usage

1. Launch the app
2. Click "Run Latency Test"
3. Click "Enable Latency Fix"
4. Follow recommendations for best results

## Notes

* Bluetooth devices may introduce high latency
* Monitor audio (NVIDIA/HD Audio) is usually slower than motherboard or USB audio

## Requirements

* Windows 10/11 64 bit (This also works on Windows 7 and 8/.1, but it's not recommended)
* .NET Desktop Runtime
